## 📋 Problem Summary

> Apex Manufacturing needs a secured, VLAN-segmented enterprise network for three departments — **Sales (VLAN 10)**, **Engineering (VLAN 20)**, and **Admin (VLAN 30)**. We design and implement this in GNS3, configure inter-VLAN routing, DHCP, ACL policies, simulate a DoS attack from Kali Linux, capture traffic in Wireshark, and demonstrate live mitigation.
